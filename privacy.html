storeAI 商品工具 隱私權政策

最後更新日期：2026-07-07

一、適用範圍

本隱私權政策說明「storeAI 商品工具」Chrome 擴充功能（以下稱「本擴充功能」）如何蒐集、使用及保護使用者資料。本擴充功能供使用者將樂天市場、Naver、淘寶／天貓等平台的商品資訊，快速轉換為 EasyStore／storeAI 商店的上架資料。

二、我們蒐集的資料

本擴充功能僅在使用者主動操作時運作，會處理以下資料：

  • 商品頁面內容：使用者開啟支援的商品頁面並點擊擴充功能時，讀取該頁面上公開顯示的商品標題、價格、圖片網址、規格等文字與圖片內容，用於產生上架資料。
  • 驗證資訊：使用者於設定頁自行輸入的 EasyStore／storeAI 商店 API Token，以及 AI 服務（Google Gemini／OpenAI／Anthropic）API Key，用於呼叫對應服務完成上架與文案生成。
  • 使用者設定：匯率、運費、最低利潤等使用者自訂的定價參數，以及每日匯率快取。

三、資料的使用方式

上述資料僅用於實現本擴充功能的單一用途——協助使用者將商品資訊上架至自己的商店，具體用途如下：

  • 商品頁面內容：用於自動填入商品標題、描述、圖片與定價欄位，並可能傳送至翻譯服務（Google 翻譯 API）進行語言轉換，以及傳送至 AI 服務（Google Gemini／OpenAI／Anthropic）以生成社群貼文文案。
  • 商品圖片：可能透過圖床服務（catbox.moe）暫存中轉，以取得可公開存取的圖片網址，供建立商店商品時使用。
  • 驗證資訊（API Token／API Key）：僅用於呼叫對應服務的 API 以完成商品上架或文案生成，不會用於其他目的。
  • 匯率資料：向台灣銀行（rate.bot.com.tw）或 ExchangeRate-API（open.er-api.com）查詢即時匯率，用於計算商品定價。

四、資料儲存方式

使用者設定與 API Token／API Key 皆儲存於使用者裝置本機的瀏覽器儲存空間（chrome.storage.local），不會上傳或同步至 StoreAI 的任何伺服器。開發者無法存取使用者儲存於本機的 Token 或 API Key。

五、資料分享與第三方服務

我們不會將使用者資料販售或轉移給第三方，亦不會將資料用於與本擴充功能單一用途無關的目的，或用於判斷信用評等、核貸等用途。

為完成上架、翻譯、定價與文案生成功能，本擴充功能會依使用者操作，將必要資料傳送至下列第三方服務：

  • EasyStore／storeAI：建立商店商品資料
  • Google 翻譯 API（translate.googleapis.com）：商品標題與描述翻譯
  • Google Gemini／OpenAI／Anthropic：AI 文案生成
  • catbox.moe：商品圖片暫存中轉
  • 台灣銀行／ExchangeRate-API：即時匯率查詢

六、使用者權利

使用者可隨時於擴充功能設定頁清除已輸入的 API Token／API Key，或直接於 Chrome 移除本擴充功能，即可刪除本機儲存的所有相關資料。

七、政策異動

本政策如有更新，將公布於本頁面，並更新上方修訂日期。建議使用者定期查閱本頁以掌握最新內容。
